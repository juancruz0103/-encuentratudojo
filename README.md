# 🥋 EncuentraTuDojo

Plataforma web y móvil orientada a la búsqueda de escuelas de artes marciales de forma rápida, organizada e intuitiva.

---

## 📌 Descripción

**EncuentraTuDojo** nace con el objetivo de conectar alumnos con escuelas de artes marciales mediante una plataforma moderna donde los usuarios puedan:

✅ Buscar escuelas cercanas  
✅ Filtrar por disciplina  
✅ Ver horarios, profesores y servicios  
✅ Consultar eventos y torneos  
✅ Explorar escuelas desde mapa interactivo  

Además, las escuelas podrán registrarse y promocionar sus actividades dentro del sistema.

---

## 🎯 Objetivo del Proyecto

Crear una solución específica para el rubro marcial que supere búsquedas genéricas como Google Maps, ofreciendo filtros especializados y una comunidad enfocada exclusivamente en artes marciales.
